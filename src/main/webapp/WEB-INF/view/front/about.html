关于本站

所用技术

本站利用后端采用SSM（Spring、Spring MVC、Mybatis），前端借助Layui框架搭建的布局，数据库采用MySQL。

部署过程

  1. 在https://github.com/ItsJingDai/JingdaiBlog-Front上下载本项目。
  2. 用Idea把项目打开。
  3. 把项目中的sql文件运行，可以在cmd窗口中利用source xxx.sql命令进行运行。
  4. 把项目中db.properties文件中的password和username改成本地的用户名和密码。
  5. 启动项目，在浏览其中输入localhost:port即可访问。

最后

由于博主的水平确实有限，项目中肯定存在很多问题，欢迎大家一起讨论学习进步。解决问题的过程也是进步的过程嘛。
